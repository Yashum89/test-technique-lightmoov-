# test-technique-lightmoov-

> Système : LightMoovFXS23 v1.0.7

## À propos

Solution complète du test technique portant sur le système 
d'éclairage automatisé LightMoovFXS23, commandé par un 
détecteur de mouvement infrarouge.

## Contenu de la solution

| Partie | Description |
|--------|-------------|
| Partie 1 | Définition des cas de tests manuels  |
| Partie 2 | Automatisation en Python  |
| Partie 3 | Analyse des logs et rédaction du bug ticket |

## Compétences appliquées

- Test boîte noire au niveau système
- Automatisation Python
- Rédaction de bug tickets
